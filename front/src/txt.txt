header.js: Обработка событий для меню-бургера и общие функции для работы с заголовком.
navigation.js: Функции для навигации и отображения различных секций сайта.
auth.js: Функции для аутентификации пользователя, включая показ и скрытие форм входа/регистрации, проверку токенов.
dataHandling.js: Все функции для обработки данных, включая запросы к API для обновления или получения данных о складе и работниках.
formHandling.js: Функции для обработки форм, включая отправку данных о товарах и регистрацию работников.
tableManipulation.js: Функции для работы с таблицами, включая редактирование, удаление и обновление данных в строках таблицы.
animations.js: Функции анимации для текста, изображений и списков.
helpers.js: Вспомогательные функции, такие как форматирование дат.



_______________
toggleEditDeleteButtons(id, isStockTable, isStaffTable): Функция для переключения видимости кнопок редактирования и удаления для конкретной строки таблицы. Принимает идентификатор строки, флаг для определения, принадлежит ли строка таблице склада или сотрудников.

________________________
saveEdit(id): Сохраняет изменения после редактирования строки в таблице склада, отправляя обновленные данные на сервер.
updateItemOnServer(id, updatedItem): Отправляет PATCH запрос на сервер с обновленными данными по конкретному элементу таблицы.

________________________



cancelEdit(id): Отменяет процесс редактирования, восстанавливая исходное отображение данных строки без сохранения изменений.

deleteItem(id): Удаляет элемент из таблицы склада, отправляя запрос на удаление на сервер и обновляя таблицу.

removeTableRow(id): Удаляет строку из DOM после успешного удаления элемента с сервера.

updateRowNumbers(): Обновляет порядковые номера строк в таблице после удаления одной из них.

startEditingStaff(id): Начинает процесс редактирования для строки таблицы сотрудников, аналогично функции startEditing.

saveWorkerEdit(id): Сохраняет изменения после редактирования строки в таблице сотрудников.

updateWorkerData(id, updatedWorkerData): Отправляет обновленные данные сотрудника на сервер для сохранения изменений.

staffCancelEdit(id): Отменяет редактирование строки сотрудника, возвращая исходное состояние строки.

deleteStaff(id): Удаляет сотрудника из таблицы, отправляя запрос на удаление на сервер.

removeWorkerRow(id): Удаляет строку сотрудника из таблицы после успешного удаления с сервера.

updateWorkerTableRow(id, updatedData, isEditable = false): Обновляет данные в строке таблицы сотрудников после успешного сохранения изменений.
